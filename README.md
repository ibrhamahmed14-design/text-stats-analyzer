# مشروعي الأول لتحليل النصوص (Text Stats Analyzer)

هذا أول مشروع لي في رحلة تعلم هندسة البيانات. ركزت فيه على بناء هيكلية منظمة للمشروع وتعلم أساسيات بيئة العمل في بايثون بدلاً من التركيز على تعقيد الكود.

## ليه عملت المشروع ده؟
كنت محتاج أطبق عملياً مفاهيم:
- تنظيم الكود في Modules.
- التعامل مع الـ venv عشان أعزل بيئة العمل.
- ترتيب المجلدات عشان يكون المشروع جاهز للرفع على GitHub والتعامل معه كمهندس بيانات مستقبلاً.
